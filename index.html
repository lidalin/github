1. 创建仓库repository

创建仓库create repository ->生成github pages Settings->Launch automatic page generator

2. 安装Git，默认设置

3. IDE配置GitHub帐号密码

4. 克隆仓库到本地

.md（README.md）文件支持

编辑.md文件->IDE安装markdown插件
注意：默认是用gbk，需要修改成utf8格式，否则中文乱码。
->更多markdown语法(http://www.baidu.com/s?wd=markdown&ie=UTF-8)

.ignore

Git使用.gitignore文件来决定哪些文件或目录被忽略
如不同步ThinkPHP中的Runtime目录
Runtime/
如不同步各种java包
*.jar
如配置文件config.php等
config.php

5. 修改文件后保存更新

add->commit->push
= commit amd push

利用GitHub部署静态页面

创建仓库->生成github pages
Settings->Launch automatic page generator->continue
会得到"http://lidalin.github.io/***",就可以访问。
默认分支：gh-pages，而不是master。
