2. 安装Git，默认设置

3. IDE配置GitHub和Git

4. 克隆仓库到本地

VCS->Checkout from Version Control->GitHub

5. 修改文件后保存更新

add->commit->push
= commit amd push

.md（README.md）文件支持

编辑.md文件->IDE安装markdown插件
注意：默认是用gbk，需要修改成utf8格式，否则中文乱码。
->更多markdown语法

.ignore

Git使用.gitignore文件来决定哪些文件或目录被忽略
如不同步ThinkPHP中的Runtime目录
Runtime/
如不同步各种java包
*.jar
如配置文件config.php等
config.php

如何利用GitHub部署静态页面

创建仓库->生成github pages
Settings->Launch automatic page generator->continue
会得到"http://lidalin.github.io/***",就可以访问。
一定要先生成GitHub Pages，这样会有一个默认分支：gh-pages，而不是master。
删除仓库也是在settings里面

直接查看GitHub静态页面

"http://htmlpreview.github.io/?"+GitHub静态页面url
如： http://htmlpreview.github.io/?https://github.com/lidalin/css/blob/gh-pages/demo3.html
